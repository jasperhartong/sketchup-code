## Key method: Entities#add_dimension_linear

**URL:** https://ruby.sketchup.com/Sketchup/Entities.html#add_dimension_linear-instance_method

Adds a linear dimension. Overloads:

- `add_dimension_linear(start_pt_or_entity, end_pt_or_entity, offset_vector)` — point/entity refs
- `add_dimension_linear(start_instance_path, end_instance_path, offset_vector)` — InstancePath + Point3d (SketchUp 2019+)
- `add_dimension_linear(start_array, end_array, offset_vector)` — array of [InstancePath or entity path, Point3d] (SketchUp 2019+)

Parameters: start and end can be `Geom::Point3d`, entity, or `[InstancePath, Geom::Point3d]`; `offset_vector` is `Geom::Vector3d` (offset from reference line to dimension line). Returns `Sketchup::DimensionLinear`.
